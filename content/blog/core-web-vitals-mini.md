## The quick answer

Most sites improve CWV by fixing images, scripts, and layout shifts.

## Quick fixes

- LCP: compress hero image, reduce render-blocking scripts
- INP: remove heavy JS, defer third-party scripts
- CLS: set image/video dimensions and avoid late-loading banners
